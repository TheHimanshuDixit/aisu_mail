## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Open the .env

```
  add necessary details Port, DB(database url) and json secret key.
```

Start the server

```bash
  npm run start
```
